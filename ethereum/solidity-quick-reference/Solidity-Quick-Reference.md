# Solidity Quick Reference
`statically typed`

`=`: assignment
`==`: equality comparison
`===`: strict equality comparison (type and value)
`!=`: inequality comparison
`!==`: strict inequality comparison (type and value)
`&&`: logical AND
`||`: logical OR
`!`: logical NOT
